# Hocamı Yıldızla (Rate My Professor)
A web app where university students can rate and review professors to help other student choose a class.

## Features
- Search for professors
- Rate professors (1-5 stars with average rating)
- Leave comments
- Add new professors

## Technologies
- Python/Flask
- SQLite
- Bootstrap
- HTML/CSS

## How to run
1. pip install flask
2. flask run
3. Open localhost:5000 on browser
